Search tips

  • All words entered in the text field are search terms
  • "Wildcard" searches: the "?" (one character) and "*" (any number of characters) wildcards are supported
  • Boolean operators: "all the words" (AND), "any word" (OR) or "exact phrase" can be selected in the query options
  • Result options: selecting "off" hides the keywords in context, while still showing a list of retrieved texts
